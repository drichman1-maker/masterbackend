## Key Features Explained

### Boundary Wire vs Vision
- **Wire**: More reliable, cheaper, requires installation
- **Vision/Camera**: Easier setup, can miss edges, pricier

### Slope Handling
- **Flat yards**: Any model works
- **Moderate hills (20-30%)**: Most mid-range models
- **Steep slopes (30-45%)**: Need all-wheel drive

### Smart Features Worth Paying For
- **GPS tracking**: Know where your mower is
- **Cellular connectivity**: Control from anywhere
- **Weather integration**: Skip mowing when raining
- **Theft protection**: GPS tracking + alarm

## Installation Requirements

**Boundary wire models:**
- 2-4 hours initial setup
- Wire must be buried or pegged
- One-time installation

**Vision/camera models:**
- 30-60 minutes setup
- No wire needed
- May need multiple training runs

## Maintenance

**Monthly:**
- Clean undercarriage
- Check blade sharpness

**Seasonal:**
- Replace blades (every 1-2 months)
- Winter storage for northern climates
- Software updates

**Annual cost: $50-150**
