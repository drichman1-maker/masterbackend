## RULES OF ENGAGEMENT

### 30-Minute Sprint Rules
1. **Set timer** — Phone/computer alarm at 30 min
2. **Commit at 25 min** — Wrap up, commit code
3. **Stop at 30 min** — Even if mid-task, document status
4. **No exceptions** — If it takes >30 min, it's a blocker

### Blocker Protocol
If any task exceeds 30 minutes:
1. Document exact error/issue
2. Decide: Continue (if 80% done) or Pivot
3. If Pivot: Move to next block, backlog the blocker
4. Revisit blockers at end of day
