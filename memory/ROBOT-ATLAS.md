# Robot Atlas - Feb 16, 2026

## About
Robot Atlas is a price + autonomy index aggregator for consumer robotics, tracking:
- Robot vacuums
- Lawn mowers
- Pool cleaners
- Window cleaners
- Kitchen robots
- Personal assistant robots

## Key Features
- Autonomy Index (RAI) scoring
- Real-time price tracking
- Product comparison engine
- Intent-based search
- Category hubs

## Stack
- Next.js (TypeScript)
- TailwindCSS
- Mock data (for now)
- Dark theme default
